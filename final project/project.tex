1. Henon Map

\begin{equation}
\begin{cases}
x_{n+1}=1-a x_n^2+y_n\\
y_{n+1}=b x_{n}
\end{cases}
\end{equation}

\vspace{1em}

1.1 Dissipative system 
\begin{equation}
J = \begin{vmatrix}
    -2ax_n & 1 \\ b & 0
\end{vmatrix}
\end{equation}

|det J| = b 일때, -1 < b <1 이라면 넓이가 감소하기 때문에 소산계(dissipative system)이다.

\vspace{1em}

1.2 Fixed points

해논 맵에서 각 고정점을 구하면
\begin{equation}
    \begin{cases}
        x^* = 1-a (x^*)^2 + y^* = 1 - a (x^*)^2 + b x^*\\
        y^* = b x^*
    \end{cases}
\end{equation}

식 (3)을 정리하여 \(x^*\)을 구하면

\begin{equation}
    x^* = \frac{(b-1) \pm \sqrt{(1-b)^2 + 4a}}{2a}
\end{equation}

이므로 \((x^*, y^*)\)을 구하면

\begin{equation}
    (x^*_{\pm}, y^*_{\pm}) =-(\frac{(b-1) \pm \sqrt{(1-b)^2 + 4a}}{2a}, b\frac{(b-1) \pm \sqrt{(1-b)^2 + 4a}}{2a})
\end{equation}

이때, 실수 고정점이 존재하기 위해서는 판별식이 0 이상이어야 함으로
\begin{equation}
    (1-b)^2 + 4a \ge 0
\end{equation}

\begin{equation}
    \therefore a \ge \frac{-(1-b)^2}{4}
\end{equation}

이다. 

\vspace{1em}

이제 선형성을 통하여 분석을 하자면 식 (2)의 야고비안에서 고정점을 대입하여 행렬식을 구하면 
\begin{equation}
J = \begin{vmatrix}
    -2ax^* - \lambda & 1 \\ b & -\lambda
\end{vmatrix}
\end{equation}

이를 통하여 구한 특성 방정식은 
\begin{equation}
    \lambda^2 + 2ax^*\lambda -b = 0
\end{equation}
이며, 고정값은
\begin{equation}
    \lambda_{\pm} = -ax^* \pm \sqrt{a^2(x^*)^2 + b}
\end{equation}
이다.

\vspace{1em}

여기서 나온 두 고유값에 대하여 \(|\lambda_1|<1,\  |\lambda_2|<1\)이라는 조건을 만족해야 함으로 b의 조건 또한 \(-1<b<1\)이라는 조건과 
\(|2ax^*| < 1-b\)의 조건을 만족 해야한다.

\vspace{1em}

\(D = \sqrt{(1-b)^2 + 4a}\)라 할때 식 (4)를 이용하여 \(2a x^*_{\pm} = b-1 \pm D\)를 구할 수 있다.
이때 b가 \(-1< b < 1\)에서 \(x^*_{-}\)는 항상 불안정 하다. 반면 \(x^*_+\)는 \(\frac{-(1-b)^2}{4} < a < \frac{3(1-b)^2}{4}\)
일때 안정이다. 

\vspace{1em}

2. Analysis of the Dynamics near an Undefined Fixed Point via the Blow-Up Technique

\vspace{1em}

비선형계에서 위상 공간을 해석하기 위해서는 위상 공간 위 벡터장이 연속이며, 미분가능해야 하며, 이는 해의 존재성과 유일성을 보장한다.
그러나 정의되지 않은 특이점이 있는 경우 특이점 근방 또는 특이점에 대한 동역학의 해석은 일반적인 해석을 적용하기 어렵다. 
본 프로젝트는 특이점 근처에서의 동역학을 Blow-Up 기법을 이용하여 원 또는 구와 같은 공간으로 확장하여 벡터장의 위상 구조를 분석하고자 한다.

\vspace{1em}

비선형계에서 위상공간을 해석하기 위해서는 벡터장이 연속이면서 미분                      이 경우에 대해서 해의 존재성과 유일성을 보장한다. 그리고 이깨의 해를 조정점이라 하며, 이 고정점에서 벡터장을 수렴하거나, 발산하는 형태 등 다양한 동역학을 가진다.
그러나 이 고정점이 저의가 되지 않은 경우 특이저이라 하는데 이 경우에는 기존 동역학으로 햐ㅐ석을 하는것에 있어서 한계가 있다.
그렇기에 이 프로젝트에서 Blow-Up이라는 기법을 이용하여 직교좌표계에서 극좌표게 또는 구면 좌표계로 변환 후 특이점에 대한 분석을 하고자 한다.

공간을 정의하면
\begin{equation}
    \begin{cases}
        \dot{x} = P(x, y) = P_{m}(x, y) + ...\\
        \dot{y} = Q(x, y) = Q_{m}(x, y) + ...
    \end{cases}
\end{equation}

이때 P와 Q는 서로소 다항식이며, \(P_{m}\), \(Q_{m}\) 은 등방성 다항식이며 이때 \(m \in \mathbb{N}\)이다.
이때 m>0이라는 점에서 특이점은 원점으로 가정한다.
이후, 극좌표를 취하면
\begin{equation}
    \begin{cases}
        \dot{r} = R(\theta)r + \dots\\
        \dot{\theta} = F(\theta)+\dots
    \end{cases}
\end{equation} 
이며, 여기서 R과 F는 \(cos(\theta), sin(\theta)\)에 대한 다항식이다.

\vspace{1em}

여기서 2가지로 가정을 할 수 있다. non-dicritical singular point라 하는 특정 방향으로 들어오는 경우이며, \(F \not\equiv 0\)일때를 말한다.
다른 표현으로는 characteristic polynomial 즉, 특성 다항식이라고 하여 \(F(\theta) = 0\)일때를 말한다.
이 경우에는 모든 방향이 아닌 특정 방향으로만 특이점에 도달하며, 특정 방향(\(\theta^* \in [0, 2\pi)\))을 characteristic direction, 특성 방향이라 한다.

\vspace{1em}

직교좌표계에서는 F를 다음처럼 쓸 수 있다.
\begin{equation}
    F(x, y) := xQ_{m}(x, y) - yP_{m}(x, y)
\end{equation} 

다른 한가지 가정은 \(F=0\)이라면 원점은 dicritical singular point이라 하며 식 (3)으로 부터
\begin{equation}
    P_{m} = xW_{m-1},\qquad Q_{m} =yW_{m-1}
\end{equation}
을 얻는다. 여기서 \(W_{m-1} \not\equiv 0\)은 차수 m-1인 등차 다항식이다. 
만약 \(y-vx\)가 \(W_{m-1}\)의 인수이고, \(v = \tan{\theta^*}, \theta^* \in [0, 2\pi)\) 이면, \(\theta^*\)는 singular direction 즉, 특이 방향성이다.

이제 두 가지 유형으로 분석 하고자 한다. 첫번째는 \(F \not\equiv 0\)이다.
이 경우는 앞서 말했듯이 한 점에 대해서 특정 방향에서 들어오는 non-dicritical singular point 이다.
이 경우에는 원점에서의 해곡선은 아무 방향을 가질 수는 없다. 즉, 특정한 방향 \(F(\theta)=0\)을 만족해야 한다는 것이며, 이때 \(F(\theta^*)=0\)을 만족하는 \(\theta^*\)를 특성 방향이라 한다.
